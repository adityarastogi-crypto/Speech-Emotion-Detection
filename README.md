# Speech-Emotion-Detection
This project focuses on recognizing human emotions from speech signals using Deep Learning techniques. The system analyzes audio inputs and predicts emotions such as happy, sad, angry, neutral, fear, and surprise based on vocal features and speech patterns.
•	Technologies: Python, TensorFlow/Keras, Librosa, CNN, NumPy, Scikit-learn  
•	Developed a deep learning model to identify and classify human emotions (e.g., happy, sad, angry) from audio recordings using Convolutional Neural Networks (CNN).  
•	Extracted MFCC (Mel-frequency Cepstral Coefficients) and chroma features from raw audio data to transform sound waves into a format suitable for neural network training.  
•	Achieved an accuracy of 94% by implementing data augmentation techniques like noise injection and pitch shifting to handle varying acoustic environments.  
•	Authored a supporting research paper detailing the methodology, feature engineering, and performance metrics of the model..
